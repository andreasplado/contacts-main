# Kontaktid

Kontaktide proovitöö.

## Kirjeldus

Lihtne proovitöö Angularis ja Spring Boots.

## Kuidas käima panna


### Installimine

#### Käivita frontend kaustas
```
ng serve
```
#### käivita backend kaustas
```
mvn spring-boot:run
```

Anmed on h2 andmebaasis ja igal uuel käivitamisel kustuvad andmed
